OpenNumismat

Το OpenNumismat είναι μια εύχρηστη και αξιόπιστη εφαρμογή που απευθύνεται σε λάτρεις νομισμάτων, νομισματολόγους ή ερασιτέχνες που γυρεύουν να δημιουργήσουν μια νομισματική συλλογή.

Με το OpenNumismat θα είστε σε θέση να οργανώσετε και να διαχειριστείτε συλλογές νομισμάτων: δημιουργήστε έναν προηγμένο κατάλογο των νομισμάτων σας, λίστες επιθυμίας και πώλησης και δείτε στατιστικά στοιχεία για τη συλλογή σας.

OpenNumismat Mobile

Σε αυτή τη σελίδα μπορείτε να δείτε τη συλλογή σας οπουδήποτε από οποιοδήποτε πρόγραμμα περιήγησης, συμπεριλαμβανομένων των κινητών (iPhone, Android).

Σημείωση ασφαλείας: Το αρχείο σας δεν θα αποσταλεί στο διαδίκτυο. Μπορείτε να απενεργοποιήσετε τη σύνδεση στο διαδίκτυο μετά το άνοιγμα της σελίδας.

Works without internet connection and supports installation as an application.

OpenNumismat for Android

Με αυτή την εφαρμογή, η συλλογή νομισμάτων σας θα είναι πάντα στο τηλέφωνό σας. Μπορείτε να χρησιμοποιήσετε έτοιμους καταλόγους ή να δημιουργήσετε τον δικό σας κατάλογο με την έκδοση του OpenNumismat για υπολογιστή.
