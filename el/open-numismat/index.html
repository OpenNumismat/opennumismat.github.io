OpenNumismat

Περιγραφή

Το OpenNumismat είναι μια εύχρηστη και αξιόπιστη εφαρμογή που απευθύνεται σε συλλέκτες νομισμάτων, νομισματολόγους ή ερασιτέχνες που γυρεύουν να δημιουργήσουν μια νομισματική συλλογή.

Με το OpenNumismat θα είστε σε θέση να οργανώσετε και να διαχειριστείτε συλλογές νομισμάτων: δημιουργήστε έναν προηγμένο κατάλογο των νομισμάτων σας, λίστες επιθυμίας και πώλησης και δείτε στατιστικά στοιχεία για τη συλλογή σας.

Στιγμιότυπα

Περισσότερα στιγμιότυπα και βίντεο

Κύρια στοιχεία

  • More than 80 customizable fields to describe the coin:
    • Χαρακτηριστικά
    • Κατάσταση του νομίσματος
    • Λεπτομέρειες έκδοσης
    • Αγορά, πώληση, μετάβαση σε πληροφορίες δημοπρασίας
    • Τιμές και αριθμοί καταλόγων
    • Address and point on the map
  • Up to 11 photos of a coin (insert image from file, clipboard, download from the Web at URL)
  • Ομαδοποίηση, φίλτρα και ταξινόμηση για τη διευκόλυνση ανάκτησης των νομισμάτων στον κατάλογο
  • Δημιουργία και εκτύπωση αναφορών και αποθήκευση ως HTML, PDF, MS Word
  • Εξαγωγή προσαρμοσμένων λιστών ως MS Excel, HTML και CSV
  • Αντίγραφο νομισμάτων για γρήγορη προσθήκη παρόμοιου νομίσματος
  • Μαζική επεξεργασία νομισμάτων
  • Built-in Image Editor
  • Webcam support for taking a photo or scan a barcode (for better results try Camo)
  • Στατιστικά
  • Search by image
  • Δυνατότητα προσθήκης και προσαρμογής των λιστών για εμφάνιση των απαιτούμενων δεδομένων
  • Search and import coin, banknote, stamp info from Colnect, an online collectors community
  • Search and import coin, medal info from ANS collections database
  • Εισαγωγή υφιστάμενων συλλογών από:
    • Excel
    • Colnect (from lists)
    • Numista
    • CoinManage (required Microsoft Access Database Engine, Microsoft Access or any other software with ODBC Microsoft Access Driver)
    • Collection Studio 3.65 (από μορφή εξαγωγής XML Table)
    • uCoin.net
    • Tellico
  • Support languages: English, Russian, German, French, Ukrainian, Spanish, Hungarian, Portuguese, Greek, Czech, Italian, Catalan, Dutch, Bulgarian, Latvian, Swedish, Persian
  • Cross-platform: Windows, Linux, macOS and Android. OpenNumismat Mobile for any browser, including mobile (iPhone, Android)
  • Δεν απαιτείται πρόσθετο λογισμικό για την εργασία με μια βάση δεδομένων
  • Αναλυτικό εγχειρίδιο χρήστη
  • Suitable for other collectibles, like stamps, banknotes, postcards, badges, scale models. And also for organize home inventory

Στοιχεία

Το OpenNumismat είναι:

OpenNumismat based on PySide6 module with SQLite database engine to store data collection.

Icons used in UI based on LED Icon Set v1.0, Soft Scraps Icons by deleket and Flag and Silk icons.

Όλες οι εικόνες και περιγραφές των νομισμάτων είναι δείγματα συλλογών από την Επίσημη ιστοσελίδα της Ευρωπαϊκής Επιτροπής, την Ιστοσελίδα της Κεντρικής Τράπεζας της Ρωσικής Ομοσπονδίας, την Ιστοσελίδα δημόσιων πληροφοριών του Νομισματοκοπείου Ηνωμένων Πολιτειών και το Wikipedia.
