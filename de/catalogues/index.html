Kataloge

Mit einem Import von Colnect (aus Listen und direkt) können Sie schnell Ihren eigenen Katalog erstellen.

Um die Arbeit mit OpenNumismat zu beginnen, kann man eine bereits erstellte Sammlung nutzen, die Informationen zu bereits produzierten Münzen enthält. (Kataloge)

    Hinweis: In Katalogen in verschiedenen Sprachen haben alle Felder auch sprachspezifische Namen. Um die Feldnamen auf Deutsch zu übersetzen, müssen Sie im Dialog Einstellungen auf der Registerkarte Feld auf die Schaltfläche "Standard wiederherstellen" klicken.
