FAQ

Mein Betriebssystem blockiert die Anwendung

Für Windows 10/11 Anwender, lesen Sie How to Fix “This App Has Been Blocked by Your System Administrator”.

Für macOS Anwender lesen Sie Safely open apps on your Mac and How to Fix “App Is Damaged and Can’t Be Opened”.

Mein Antivirusprogramm meldet eine Schadsoftware

Warten Sie 1-2 Tage, bis die Virendatenbanken aktualisiert sind, oder wenden Sie sich an den Hersteller des Betriebssystems oder des Virenschutzprogramms (für Microsoft Security Essentials: Submit a file for malware analysis, for Norton: Norton Submission Portal).

Wo kann ich die Version für Android herunterladen?

Die Android-Version wird derzeit nicht unterstützt, aber Sie können Folgendes verwenden: OpenNumismat Mobile. Funktioniert ohne Internetverbindung und unterstützt die Installation als Anwendung.

Ist eine iOS/iPhone/iPad App geplant?

Vielleicht in ferner Zukunft, aber Sie können OpenNumismat Mobile verwenden. Funktioniert ohne Internetverbindung und unterstützt die Installation als Anwendung.

Ich habe die Demo-Version heruntergeladen. Wo kann ich die Vollversion bekommen?

Alle Versionen von OpenNumismat sind Vollversionen. Die Bezeichnung „Demo“ rührt daher, dass dies der Name der einfachsten, mit der Anwendung mitgelieferten Sammlung ist.

Sie können die Sammlungsdatei nach Belieben umbenennen (über Ihr Betriebssystem) und diese verwenden oder eine neue Sammlung erstellen. Überprüfen Sie auch die verfügbaren Kataloge.

Ich sammle Banknoten, Briefmarken, usw. Kann OpenNumismat für meine Sammlung angepasst werden?

Ja, OpenNumismat kann für jede Art von Sammlung verwendet werden. Nutzer verwenden es für das Sammeln von Postkarten, Autos oder sogar Veilchen. In den Einstellungen müssen dazu manche Felder umbenannt werden.

Wo kann ich vollständige Kataloge für meine Sammlung erhalten?

Sie können einige Kataloge unter finden. Dies ist jedoch kein kommerzielles Projekt, und es gibt keine Möglichkeit, Kataloge auf dem neuesten Stand zu halten.

Wenn Sie einen fertigen Katalog haben und ihn mit anderen teilen möchten, senden Sie ihn an opennumismat@gmail.com.

Außerdem können Sie Informationen zu Münzen, Briefmarken oder Banknoten mit Bildern von der Colnect Online Datenbank mit mehr als 175.000 Münzen importieren. Oder Informationen zu Münzen und Medaillen mit Bildern von derANS Sammlungsdatenbank mit mehr als 160.000 Münzen.

Wenn Sie bereits ein Konto bei Colnect oder Numista haben, können Sie Ihre Sammlung einfach importieren.

Ich benötige einen Katalog mit einem automatischen Update des aktuellen Marktpreis meiner Münzen

OpenNumismat verfügt über keine Kataloge mit Marktpreisen. Da es sich nicht um ein kommerzielles Projekt handelt, können die Preise nicht aktuell gehalten werden. Sehen Sie sich stattdessen andere kommerzielle Anwendungen an.

Warum gibt es so viele Felder? Das Fenster passt nicht auf den Bildschirm.

Sie können nicht verwendete Felder unter Einstellungen->Felder deaktivieren.

Wie kann ich die Anwendung an meine Bedürfnisse anpassen? Kann ich einige Felder umbenennen?

Sie können alle Felder unter Einstellungen->Felder deaktivieren oder umbenennen.

Warum ist die Marktplatzseite deaktiviert?

Um die Marktseite zu aktivieren ändern Sie den Status einer Münze auf Beobachtet, Vorhanden, Bestellt, Verkauft, Verkauf, Bieten, Fehlt oder Doppelt.

Kann ich in den Feldern Jahr, Gewicht, Durchmesser usw. Bereiche oder Zeichen eingeben?

Yes. Enable Free format for numeric fields in Collection page of Settings.

I add photos of coins, but the Image field is empty

The image field is formed from two photos - obverse and reverse from Design page. The rest of the images can be displayed separately in the coin table or viewed on the image pane.

Can I use different reference for different collections? Can I attach reference to collection?

Yes, menu Reference->Attach current reference.

Importing from Colnect or American Numismatic Society don't work

Please try clearing the import cache from the Settings->Import->Clear cache.

Die Anwendung stürzt ab. Nichts funktioniert. Ich habe den Fehler gefunden

Zuallererst: Keine Panik!

Enable Send error info to author in Settings.

If the problem is important to you, please send a detailed bug report to opennumismat@gmail.com, attach screenshots or the problem file.
