OpenNumismat

Beschreibung

OpenNumismat ist ein handliches und zuverlässiges Programm für Münzliebhaber - erfahrene Numismatiker oder Neueinsteiger - die das Ziel haben, ihre Sammlung zu dokumentieren.

MitOpenNumismat können Sie Ihre Münzsammlung verwalten: Erstellen sie einen detaillierten Katalog Ihrer Münzen; erzeugen Sie Wunsch- und Verkaufslisten und werten Sie Statistiken Ihrer Sammlung aus.

Hauptmerkmale

  • Mehr als 80 auswählbare Felder zur Beschreibung der Münze:
    • Eigenschaften der Münze
    • Zustand und Beschaffenheit der Münze
    • Ausgabedetails
    • Daten über Kauf, Verkauf, usw.
    • Preise und Katalognummern
    • Adresse / Punkt auf Landkarte
  • Bis zu 11 Fotos einer Münze (Bild aus Datei einfügen, Zwischenablage, Download aus dem Web)
  • Übersichtliches Suchen und Finden von Münzen im Katalog durch die Möglichkeit zu gruppieren, filtern und sortieren.
  • Herstellen und Ausdrucken von Berichten, die als HTML, PDF oder als MS-Word-Dateien gespeichert werden können.
  • Export benutzerdefinierter Listen in Form von MS-Excel-, HTML- oder CSV-Dateien.
  • Duplizierung von Münzen, um schnell ähnliche Münzen einzugeben.
  • Bearbeiten von Münzen im Stapelverfahren.
  • Eingebauter Bildeditor
  • Webcam-Unterstützung zum Fotografieren oder Scannen von Barcodes (für bessere Ergebnisse versuchen Sie: Camo)
  • Statistiken
  • Suche nach Bild
  • Möglichkeit, benutzerdefinierte Listen zu konfigurieren und zu speichern, um die gewünschten Daten anzuzeigen.
  • Suche und Import von Infos zu Münzen, Banknoten und Briefmarken aus Colnect, einer Online-Sammler-Community
  • Suche und Import von Infos zu Münzen und Medaillen aus der ANS -Sammlungsdatenbank
  • Import vorhandener Sammlungen aus:
    • Excel
    • Colnect (aus Listen)
    • Numista
    • CoinManage (erforderlich: Microsoft Access Database Engine, Microsoft Access oder jede andere Software mit ODBC Microsoft Access Driver)
    • Collection Studio 3.65 (aus dem XML Tabellen export)
    • uCoin.net
    • Tellico
    • CoinSnap
  • Unterstützte Sprachen: Englisch, Russisch, Deutsch, Französisch, Ukrainisch, Spanisch, Ungarisch, Portugiesisch, Griechisch, Tschechisch, Italienisch, Katalanisch, Niederländisch, Bulgarisch, Lettisch, Schwedisch, Persisch
  • Plattformübergreifend: Windows, Linux, macOS und Android. OpenNumismat Mobile für jeden Browser, auch Mobilgeräte (iOS, iPadOS, Android)
  • Für die Arbeit mit der Datenbank wird keine zusätzliche Software benötigt.
  • Ausführliches Benutzerhandbuch
  • Verwendbar für alle Sammlerstücke, wie Briefmarken, Banknoten, Postkarten, Abzeichen, Modellbau. Und auch zur Verwaltung des Hausinventars

Wie kann ich helfen?

  • Benutzen Sie OpenNumismat und empfehlen Sie es weiter.
  • Bitte melden Sie Fehler und neue Funktionen, die OpenNumismat verbessern.
  • Übersetzen Sie OpenNumismat und die Dokumentation in Ihren Sprache mit Transifex (Bitte mit der lang.ts Datei beginnen).
  • Wenn Sie über Exklusivrechte von Katalogen, Abbildungen oder anderen numismatischen Inhalten verfügen, die in das Programm aufgenommen werden können, wenden Sie sich bitte an den Autor.
  • Bei Fragen oder Anregungen können Sie den Autor kontaktieren: opennumismat@gmail.com

Details

OpenNumismat ist:

OpenNumismat based on PySide6 module with SQLite database engine to store data collection.

Die im Benutzerinterface benutzten Icons basieren auf LED Icon Set v1.0, Soft Scraps Icons by deleket und Flag and Silk icons.

Alle Abbildungen und Beschreibungen von Münzen in der Demo-Sammlung entstammen folgenden Seiten: Official website of the European Commission, The Central Bank of the Russian Federation website, United States Mint's public information Website und Wikipedia.
