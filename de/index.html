OpenNumismat

OpenNumismat ist ein handliches und zuverlässiges Programm für Münzliebhaber - erfahrene Numismatiker oder Neueinsteiger - die das Ziel haben, ihre Sammlung zu dokumentieren.

MitOpenNumismat können Sie Ihre Münzsammlung verwalten: Erstellen sie einen detaillierten Katalog Ihrer Münzen; erzeugen Sie Wunsch- und Verkaufslisten und werten Sie Statistiken Ihrer Sammlung aus.

OpenNumismat Mobile

Auf dieser Seite können Sie Ihre Sammlung mit jedem Browser betrachten, einschliesslich dem Ihres Mobilgeräts (iPhone, Android)

Sicherheitshinweis: Ihre Datei wird nicht auf einen Server geladen. Sie können die Internetverbindung nach dem Öffnen der Seite deaktivieren.

Funktioniert ohne Internetverbindung und unterstützt Installation als App.

OpenNumismat für Android

Mit diesem Programm haben Sie Ihre Münzsammlung immer mit auf Ihrem Handy. Sie können bestehende oder eigene Kataloge der Desktop-Version von OpenNumismat benutzen.
