Catalogues pour mobile

Pour la version mobile d'OpenNumismat vous pouvez utiliser des catalogues existant(aperçu disponible), optimisés pour l'écran des mobiles.

Télécharger et copier sur votre appareil mobile n'importe lequel des catalogues suivants:

    Pour choisir le fichier adapté à votre appareil, utiliser la résolution la plus proche parmi les suivantes:

    • mdpi (moyenne) ~160dpi
    • hdpi (haute) ~240dpi
    • xhdpi (extra-haute) ~320dpi
    • xxhdpi (extra-extra-haute) ~480dpi
    • xxxhdpi (extra-extra-extra-haute) ~640dpi

    Note : Tous les catalogues utilisent des informations provenant de diverses sources avec des licences différentes. Faites attention pour tout usage non privé. Tous les éléments appartiennent à leurs propriétaires. (Je le vérifie et change les données ou la licence.)
