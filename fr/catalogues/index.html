OpenNumismat

est une collection d'un programme de collection de pièces de monnaie libre et open-source

Catalogues

Pour commencer à travailler avec OpenNumismat vous pouvez utiliser une collection déjà existante qui contient toutes les informations sur les pièces émises (catalogues).

Les catalogues suivant sont disponibles :

    • Pièces standard de l'UE (Anglais) 24.04.2013
    • Pièces commémoratives ukrainiennes (Ukrainien) 19.02.2013
    • toutes les pièces russes (Russe) (Pièces standard et commémoratives 1992-2013) 08.04.2013

    Note : Tous les catalogues utilisent des informations provenant de diverses sources avec des licences différentes. Faites attention pour tout usage non privé. Tous les éléments appartiennent à leurs propriétaires. (Je le vérifie et change les données ou la licence.)

    Note : Dans les catalogues en anglais, tous les champs ont des noms anglais. Pour traduire le champs dans votre langue, vous devez aller dans l'onglet Paramètres du champ et appuyez sur le bouton Remettre par défaut.
