Catalogues

Par l'importation de Colnect (depuis listes et directement) vous pouvez créer rapidement votre propre catalogue.

Pour commencer à travailler avec OpenNumismat vous pouvez utiliser une collection déjà existante qui contient toutes les informations sur les pièces émises (catalogues).

Les catalogues suivant sont disponibles :

    Note : Tous les catalogues utilisent des informations provenant de diverses sources avec des licences différentes. Faites attention pour tout usage non privé. Tous les éléments appartiennent à leurs propriétaires. (Je le vérifie et change les données ou la licence.)

    Note: Dans les catalogues en différentes langues tous les champs ont leurs noms propres. Traduire le nom du champs dans votre propre langue est nécessaire dans la boite de dialogue Paramétres index du Champs appuyez sur le boutonRestaurez par defaut.
