OpenNumismat

Descripción

OpenNumismat es una aplicación práctica y fiable dirigida a los amantes de las monedas, los numismáticos o aficionados que buscan crear una colección numismática.

Con OpenNumismat, podrá organizar y administrar la colección de monedas: cree un catálogo avanzado de sus monedas, listas de deseos y venta, vea las estadísticas de su colección.

Principales características

  • Más de 80 campos personalizables para describir la moneda:
    • Características
    • Estado de una moneda
    • Detalles de emisión
    • Compra venta, préstamo en información sobre subasta
    • Precios y números de catálogo
    • Direccionar y apuntar en el mapa
  • Hasta 11 fotos de una moneda (inserte la imagen desde archivo, portapapeles, descárguelo desde la URL de Internet)
  • La agrupación, filtros y clasificación para facilitar la recuperación de monedas en el catálogo
  • Genere e imprima informes, guarde como HTML, PDF, MS Word
  • Exporte listas personalizadas en MS Excel, HTML y CSV
  • Duplicación de monedas para añadir rápidamente una pieza similar
  • Edición por lotes de monedas
  • Editor de imagen integrado
  • Estadísticas
  • Búsqueda por imagen
  • Posibilidad de agregar y personalizar las listas para mostrar los datos requeridos
  • No requiere software adicional para trabajar con una base de datos
  • Busca e importa información de monedas, billetes, sellos desde Colnect, una comunidad de coleccionistas en línea.
  • Busca e importa información de monedas, medallas desde la base de datos de colecciones ANS
  • Importe colecciones existentes desde:
    • Excel
    • Colnect (desde listas)
    • Numista
    • CoinManage (necesario Motor Microsoft Access Database, Microsoft Access o cualquier otro software con el controlador ODBC Microsoft Access)
    • Collection Studio 3.65 (de una tabla con formato de exportación XML)
    • uCoin.net
    • Tellico
  • Idiomas soportados: inglés, ruso, alemán, francés, ucraniano, español, húngaro, portugués, griego, checo, italiano, catalán, holandés, búlgaro, lituano, sueco, persa
  • Multi-plataforma: Windows, Linux, macOS y Android. OpenNumismat Móvil para cualquier navegador, incluyendo móviles (iPhone, Android)
  • Manual de usuario detallado
  • Válido para otros coleccionables, como sellos, billetes, postales, escudos, modelos a escala. Y también para organizar un inventario doméstico

Detalles

OpenNumismat es una:

OpenNumismat based on PySide6 module with SQLite database engine to store data collection.

Iconos usados en UI basados en LED Icon Set v1.0, Soft Scraps Icons por deleket y Flag and Silk icons.

Todas las imágenes y descripciones de las colecciones demo provienen de la Web oficial de la Comisión Europea, la Web oficial del Banco Central de la Federación Rusa, la Web de información de United States Mint y Wikipedia.
