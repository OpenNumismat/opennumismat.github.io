OpenNumismat

Descripción

OpenNumismat es una aplicación práctica y fiable dirigida a los amantes de las monedas, los numismáticos o aficionados que buscan crear una colección numismática.

Con OpenNumismat, podrá organizar y administrar la colección de monedas: cree un catálogo avanzado de sus monedas, deseos y listas de venda, vea las estadísticas de su colección.

Principales características

  • Más de 70 campos personalizables para describir la moneda:
    • Características
    • Estado de una moneda
    • Detalles de emisión
    • Comprrar, vender, pasar información sobre subasta
    • Precios y números de catálogo
  • Subir hasta 8 fotos de una moneda (inserte la imagen desde archivo, portapapeles, descárguelo desde Internet)
  • La agrupación, filtros y clasificación para facilitar la recuperación de monedas en el catálogo
  • Generar e imprimir informe, guardando como HTML, PDF. MS Word
  • Exportar lista personalizada como MS Excel, HTML y CSV
  • Duplicar monedas para añadir rápido una pieza similar
  • Ejecutar edicción de monedas
  • Estadísticas
  • Posibilidad de agregar y personalizar las listas para mostrar los datos requeridos
  • No requiere software adicional para trabajar con base de datos
  • Importar monedas, billetes, sellos de Colnect, una comunidad de coleccionistas en línea.
  • Importar colecciones existentes desde:
    • Excel (con imágenes presentadas como enlaces en el archivo fuente)
    • Cabinet 2.2.2.1 (desde formato de exportación)
    • CoinManage 2011, 2013, 2015 (necesario Driver Visual FoxPro ODBC, Motor Microsoft Access Database, Microsoft Access o cualquier otro software con el controlador ODBC Microsoft Access Driver)
    • Collection Studio 3.65 (de una tabla con formado de exportación XML)
    • uCoin.net
    • Tellico
  • Lenguajes soportados: Inglés, Ruso, Alemán, Francés, Ucraniano, Español, Húngaro, Portugués, Griego, Checo, Italiano, Catalán, Holandés, Búlgaro, Lituano, Sueco, Persa
  • Multiplataforma: Windows, Linux, MacOS y Android. Visor en línea para cualquier navegador, incluidos los dispositivos móviles (iPhone, Android)
  • Manual de usuario detallado
  • Válido para otras colecciones, como sellos, billetes, postales, divisas

Detalles

OpenNumismat es una:

OpenNumismat basado en PyQt framework con motor SQLite database para almacenar datos de la colección.

Iconos usados en UI basados en LED Icon Set v1.0, Soft Scraps Icons por deleket y Flag and Silk icons.

Todas las imágenes y descripciones de las colecciones demo provienen de Web oficial de la Comisión Europea, Web oficial del Banco Central de la Federación Rusa, Web de información de United States Mint y Wikipedia.
