FAQ

¿Dónde puedo conseguir la versión para Android?

Android version is currently not supported, but you can use OpenNumismat Mobile. Works without internet connection and supports installation as an application.

Is an iOS/iPhone/iPad app planned?

Quizá en un futuro distante, pero puedes usar OpenNumismat Móvil. Funciona sin conexión a internet y soporta la instalación como aplicación.

Descargué la versión demo. ¿Dónde puedo obtener la versión completa?

Todas las versiones de OpenNumismat son completas. Ve "demo" porque es el nombre de la colección más simple que viene con la aplicación.

Puede renombrar el archivo de colección como quiera (usando su SO) y usarlo para crear una nueva colección. Revise también los catálogosdisponibles.

Colecciono billetes, sellos, etc. ¿Se puede adaptar OpenNumismat para mi colección?

Sí, puedes usar OpenNumismat para cualquier colección. Hay usuarios que lo usan para coleccionar postales, coches o incluso violetas. Necesitarás renombrar algunos campos en Configuración.

¿Dónde puedo encontrar catálogos completos para mi colección?

Puedes encontar algunos catálogos aquí. Pero no es un proyecto comercial y no hay forma de mantener los catálogos actualizados.

Si tienes un catálogo completo y lo quieres compartir con otros, envíalo a opennumismat@gmail.com.

Also you can import coin, stamp or banknote information with images from Colnect's online database with more than 170.000 coins. If you already have account on Colnect you can simply import all the information.

Necesito un catálogo con actualización automática de precio de mercado para mis monedas.

OpenNumismat no tiene catálogos con precio de mercado. Esto no es un proyecto comercial y no hay forma de mantener los precios actualizados. Busca otras aplicaciones comerciales.

¿Por qué hay tantos campos? La ventana no cabe en la pantalla.

Puedes desactivar los campos que no usas en Configuración->Campos.

¿Cómo puedo adaptar la aplicación para mí? ¿Puedo renombrar algunos de los campos?

Puedes desactivar o renombrar todos los campos en Configuración->Campos.

La aplicación se cuelga. Nada funciona. Encontré el error.

Para empezar: ¡nada de pánico!

Activa Enviar info de error al autor en Configuración.

Si el problema es importante para ti, por favor, envía un informe detallado del fallo a opennumismat@gmail.com, y adjunta capturas de pantalla o el archivo problemático.

Mi SO está bloquendo la app. Mi antivirus detecta un malware

Espera 1-2 días a que se actualicen las bases de datos del antivirus o contacta con el fabricante del SO o del antivirus (para Microsoft Security Essentials: Enviar archivo para análisis de malware, para Norton: Portal de Envíos Norton).

Para usuarios macOS, leer Abrir apps de forma segura en tu Mac.

¿Por qué está desactivada la página Mercado?

Para activar la página Mercado cambie el estado de la moneda a Prestada, En propiedad, Pedida, Vendida, En venta, En puja, Perdida o Duplicada.

Añado fotos de monedas, pero el campo Imagen está vacío

The image field is formed from two photos - obverse and reverse from Design page. The rest of the images can be displayed separately in the coin table or viewed on the image pane.

¿Puedo usar distintas referencias para distintas colecciones? ¿Puedo adjuntar la referencia a la colección?

Sí, menú Referencia->Adjuntar referencia actual.
