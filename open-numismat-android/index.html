OpenNumismat for Android

Description

OpenNumismat is an application for coins collectors.

With this application your coins collection always in your Android device. You can use existent catalogues or create your own catalogue with desktop version of OpenNumismat.

This is mobile version of OpenNumismat that helps track of which coins you already have.
