OpenNumismat

is a collection of free, open-source coin collecting software

Catálogos

Para começar a trabalhar com o OpenNumismat pode usar uma coleção já criada que contenha informação acerca de moedas produzidas (catálogos).

Os seguintes catálogos estão disponíveis para transferir:

    • Moedas regulares da UE (Ingês) 24.04.2013
    • Comemorativas ucranianas (Ucraniano) 19.02.2013
    • Todas as moedas russas (Russo) (comemorativas e linguotes 1992-2013) 08.04.2013

    Nota: todos os catálogos usam informação de muitas fontes com diferentes licenças. Tenha cuidado com utilizações não privadas. Todos os materiais pertencem aos seus proprietários (Nós verificamos e alteramos os dados ou a licença).

    Nota: nos catálogos em inglês todos os campos têm também o nome em inglês. Para traduzir os nomes de campos para o seu idioma, é necessário clicar em Reverter para predefinição, no diálogo Definições, separador Campos.
