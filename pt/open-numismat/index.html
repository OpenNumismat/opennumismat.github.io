OpenNumismat

Descrição

OpenNumismat é uma aplicação prática e de confiança, destinada aos amantes de moedas, numismatas ou amadores tentando criar uma colecção numismática.

Com o OpenNumismat, poderá organizar e gerir colecções de moedas: criar um catálogo avançado das suas moedas, listas de desejadas e vendidas, ver estatísticas da sua colecção.

Funcionalidades principais

  • Mais de 80 campos personalizáveis para descrever a moeda:
    • Características
    • Estado da moeda
    • Detalhes de emissão
    • Informação acerca de compra, venda, troca ou leilão
    • Preços e números de catálogo
    • Endereço e mapa
  • Até 11 imagens por moeda (a partir de ficheiro, da memória, transferidas da Internet)
  • Agrupar, filtrar e ordenar para facilitar a procura de moedas no catálogo
  • Gerar e imprimir relatórios, gravando como HTML, PDF, MSWord
  • Exportar listas personalizadas como MS Excel, HTML e CSV
  • Duplicação de moedas para adicionar rapidamente uma moeda similar
  • Editar moedas em lote
  • Editor de imagens interno
  • Estatísticas
  • Capacidade de adicionar e personalizar as listas para mostrar os dados requeridos
  • Não requer programa adicional para trabalhar com a base de dados
  • Importação de informação sobre moedas, notas e selos de Colnect, uma comunidade de coleccionadores online
  • Importa de CoinsCollector, Numizmat 2, Cabinet, CoinManage 2011, 2013, Collection Studio 3.65 e Numizmatik_Ru
    • Excel (com imagens como ligações no ficheiro fonte)
    • Colnect (das listas e directamente)
    • CoinManage (requerido Microsoft Access Database Engine, Microsoft Access ou qualquer programa com o controlador ODBC Microsoft Access Driver)
    • Numista
    • Collection Studio 3.65 (de um formato de exportação como tabela XML)
    • uCoin.net
    • Tellico
    • Cabinet 2.2.2.1 (de um formato de exportação)
  • Suporta os idiomas: Alemão, Búlgaro, Catalão, Checo, Espanhol, Francês, Grego, Holandês, Húngaro, Inglês, Italiano, Letão, Persa, Português, Russo, Sueco, Ucraniano
  • Multi-plataforma: Windows, Linux, macOS e Android. OpenNumismat Mobile para qualquer navegador, incluindo móveis (iPhone, Android)
  • Manual de utilizador detalhado
  • Adequado para outros coleccionáveis, tais como selos, notas, postais, emblemas. E também para organizar o inventário doméstico

Detalhes

O OpenNumismat é:

O OpenNumismat é baseado em PyQt framework com motor de base de dados SQLite para armazenar a colecção.

Ícones usados baseados em LED Icon Set v1.0, Soft Scraps Ícones por deleket e Flag and Silk icons.

Todas as imagens e descrições das moedas nas colecções de exemplo tiradas da Página oficial da Comissão europeia, Página Web do The Central Bank of the Russian Federation, Página Web de informação pública da Casa da moeda dos EUA e da Wikipedia.
