OpenNumismat

Descrição

OpenNumismat é uma aplicação prática e de confiança, destinada aos amantes de moedas, numismatas ou amadores tentando criar uma colecção numismática.

Com o OpenNumismat, será capaz de criar, organizar e gerir o seu catálogo de moedas com uma descrição detalhada e fotografias de cada um dos itens.

Características principais

  • Mais de 60 campos personalizáveis para descrever a moeda:
    • Características
    • Estado da moeda
    • Detalhes de emissão
    • Informação acerca de compra, venda, troca ou leilão
    • Preços e números de catálogo
  • Até 7 fotografias da moeda (inserir imagem de ficheiro, memória, transferir da web no URL)
  • Agrupar, filtrar e ordenar para facilitar a procura de moedas no catálogo
  • Gerar e imprimir relatórios, gravar como HTML, PDF, MS Word
  • Exportar listas personalizadas como MS Excel, HTML e CSV
  • Duplicação de moedas para adicionar rapidamente uma moeda similar
  • Editar moedas em lote
  • Capacidade de adicionar e personalizar as listas para mostrar os dados requeridos
  • Não requer programa adicional para trabalhar com a base de dados
  • Importa de CoinsCollector, Numizmat 2, Cabinet, CoinManage 2011, 2013, Collection Studio 3.65 e Numizmatik_Ru
    • CoinsCollector 2.6
    • Numizmat 2.1 (requerido Firebird 2.0)
    • Cabinet 2.2.2.1 (de um formato de exportação)
    • CoinManage 2011, 2013, 2015 (requerido controlador Visual FoxPro ODBC, Microsoft Access Database Engine, Microsoft Access ou qualquer outro programa com o controlador ODBC Microsoft Access)
    • Collection Studio 3.65 (de um formato de exportação como tabela XML)
    • Numizmatik_Ru 1.0.0.82
    • uCoin.net
    • Tellico
  • Suporta os idiomas: Alemão, Catalão, Checo, Espanhol, Francês, Grego, Holandês, Húngaro, Inglês, Português, Russo e Ucraniano
  • Multi-plataforma: Windows, Linux, MacOS e Android.

Detalhes

O OpenNumismat é:

O OpenNumismat é baseado em PyQt framework com motor de base de dados SQLite para armazenar os dados da colecção.

Os ícones utilizados no UI são baseados nos conjuntos LED v1.0 e Soft Scraps Icons por deleket.

Todas as imagens e descrições das moedas nas colecções de exemplo tiradas da Página oficial da Comissão europeia, Página Web do The Central Bank of the Russian Federation, Página Web de informação pública da Casa da moeda dos EUA e da Wikipedia.
