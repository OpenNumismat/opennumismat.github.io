FAQ

Onde posso obter a versão Android?

Ainda não dispomos de uma versão Android, mas pode utilizar o OpenNumismat Mobile. Funciona sem ligação à Internet e suporta instalação como aplicação.

Há planos para uma aplicação iOS/iPhone?

Talvez, num futuro distante, mas pode utilizar o OpenNumismat Mobile. Funciona sem ligação à Internet e suporta instalação como aplicação.

Transferi a versão de demonstração. Onde posso obter a versão completa?

Todas as versões do OpenNumismat estão completas. Só vê "demo" por este ser o nome da colecção simples fornecida com a aplicação. Pode renomear o ficheiro como quiser (utilizando o seu SO) e utilizá-lo, ou criar uma nova colecção. Veja também os catálogos disponíveis.

Também colecciono notas, selos, etc. Posso adaptar o OpenNumismat à minha colecção?

Sim, pode adaptar o OpenNumismat a qualquer colecção. Há utilizadores com colecções de postais ilustrados, carros e mesmo violetas. Terá apenas de renomear alguns campos nas definições gerais.

Onde posso obter catálogos completos da minha colecção?

Pode encontrar alguns catálogos aqui. Mas este não é um projecto comercial, não temos forma de manter catálogos actualizados.

Se dispõe de um catálogo completo e deseja partilhá-lo com outros utilizadores, por favor envie-o para opennumismat@gmail.com.

Preciso de um catálogo com preço de mercado automático para as minhas moedas

O OpenNumismat não tem catálogos com preços de mercado. Este não é um projecto comercial, não temos forma de manter catálogos actualizados. Procure outras aplicações comerciais.

Porque é que há tantos campos? A janela não cabe no ecrã.

Pode desactivar campos em Definições -> Campos.

Como é personalizo a aplicação para mim? Posso renomear campos?

Pode desactivar ou renomear campos em Definições -> Campos.

A aplicação rebenta. Nada funciona. Encontrei um erro.

Primeiro que tudo, não entre em pânico!

Active o envio de informações do erro para o autor nas definições.

Se o problema é importante, por favor envie também uma descrição detalhada do erro para opennumismat@gmail.com, anexe capturas de ecrã e descreva o que fazia na altura. Se possível, envie o ficheiro do erro.

O meu SO bloqueia a aplicação. O meu anti-vírus detecta uma ameaça.

Aguarde 1 a 2 dias para actualizar as bases de dados dos anti-vírus ou contacte o fabricante do SO ou anti-vírus (para Microsoft Security Essentials: Submeta u, ficheiro para análise de ameaças, para Norton: Norton Submission Portal).

Para utilizadores macOS, leiam Safely open apps on your Mac

A barra de ferramentas, ou o painel de imagem/informação desapareceram.

Adicionei fotografias da moeda, mas o campo está em branco.

O campo de imagem é formado por duas fotografias - obverso e reverso. O resto das imagens pode ser mostrado separadamente na tabela de moedas ou visto no painel de imagens.

Posso utilizar referências diferentes para diferentes colecções? Posso anexar referências às colecções?

Sim, menu Referência -> Anexar referência actual.
