OpenNumismat

is a collection of free, open-source coin collecting software

Histórico de alterações OpenNumismat

O que há de novo no OpenNumismat 1.5.3 (19 de fevereiro de 2016)

  • Reparado cursor perdido ao inserir registos
  • Capacidade de adicionar moedas de outra coleção
  • Reparada a filtragem com a coluna de estado
  • Adicionado botão Renomear campo às definições
  • Reparado campos numéricos reais para alguns idiomas Romance
  • Melhorados valores de definição para o campo Cunhagem
  • Reparada a colagem de imagens da web
  • Reparada a limpeza de campos de data
  • Adicionada a tradução francesa (por Mathieu Martineau, Jean-Claude Kursner)
  • Adicionada opção para comprimir imagens ao exportar para móvel
  • Atualizado ícone de limpeza de filtros
  • Atualilizadas bibliotecas e dependências adicionais

O que há de novo no OpenNumismat 1.5.2 (06 de fevereiro de 2015)

  • Reparado erro no idioma polaco
  • Reparado armazenamento do estado maximizado da janela

O que há de novo no OpenNumismat 1.5.1 (02 de fevereiro de 2015)

  • Reparado o erro ao gravar ordenação
  • Adicionada a importação de uCoin.net
  • Reparada a importação de CoinManage
  • Adicionado filtro para valores vazios na árvore
  • Diálogo de personalização da árvore melhorado
  • Reparada a navegação na árvore com denominação grande (>1000)
  • Reparados os campos Data e Rebordo
  • Acelerada a remoção de moedas
  • I18n

O que há de novo no OpenNumismat 1.5 (19 de dezembro de 2014)

  • Atualizado PyQt para 5.3.2 e Python para 3.4.2
  • Implementada caixa de procura no diálogo de filtragem de coluna
  • Exportar para móvel
  • Novas opções para ordenar itens no painel esquerdo em árvore e no diálogo de filtragem da coluna
  • Editor de referência melhorado, possibilidade de adicionar ícones a período
  • Novo painel do menu principal melhorado
  • Novas opções de instalação: permitir procurar atualizações e associar ficheiros .db
  • Adicionada tradução para italiano (por Teroknor)
  • Adicionada tradução para polaco (por Nazzy)

O que há de novo no OpenNumismat 1.4.9 (9 de setembro de 2014)

  • Added Czech translations (by fri)
  • Fixed debian package for Ubuntu 14.04
  • Fixed commission field processing

O que há de novo no OpenNumismat 1.4.8 (6 de agosto de 2014)

  • Adicionada a tradução para Alemão (por Detlev Vogel)
  • Adicionada a tradução para Grego(por geogeo)
  • Adicionado novo estado das moedas - Ordenada

O que há de novo no OpenNumismat 1.4.7 (11 de março de 2014)

  • Adicionado traduções para português(por Pedro Albuquerque)
  • Reparados erros reportados pelos utilizadores

O que há de novo no OpenNumismat 1.4.6 (10 de janeiro de 2014)

  • Reparados erros reportados pelos utilizadores

O que há de novo no OpenNumismat 1.4.5 (6 de junho de 2013)

  • Adicionado modelo de relatório Personalizado_1
  • Implementada importação de Numizmatik_Ru

  • Melhorada filtragem e ordenação
  • Melhorada estrutura da BD para mais rapidez

  • Reparado formato livre nos campos de preço. Reparado calculo da comissão na venda
  • Adicionado gravar tamanho do editor de referência, pré-visualização de relatório, diálogos de definições
  • Movida a configuração do tamanho da imagem da coleção para as definições globais

O que há de novo no OpenNumismat 1.4.4 (2 de abril de 2013)

O que há de novo no OpenNumismat 1.4.2 (15 de fevereiro de 2013)

  • Reparado erro ao definir a palavra-passe

  • Reparado detetar caminho dos recursos da aplicação
  • Atualizado criar ficheiros e pastas predefinidos (referência e BD demonstração predefinidos) se em falta

O que há de novo no OpenNumismat 1.4.1 (31 de janeiro de 2013)

  • Reparado erro crítico ao abrir a aplicação quando o registo está limpo (problema 12).

O que há de novo no OpenNumismat 1.4 (31 de janeiro de 2013)

  • Implementado geração de relatórios (baseado em modelos Jinja2)
  • Implementado exportar lista personalizada (Relatório> Gravar lista atual...)
  • Adicionado capacidade de introduzir quaisquer dados em campos numéricos (ativar Formato livre em campos numéricos no diálogo Definições) (problema 6, problema 10)
  • Reparado automatic update check (with frequent application startup check is not performed)
  • Adicionado comando Vácuo (menu Coleção) que liberta espaço em disco reservado por motores de BD
  • Alterado o nome de ficheiro predefinido ao gravar imagens de moedas para um ficheiro
  • Reparado possível erro ao abrir o ficheiro especificado no campo URL
  • Reparado possível erro ao ao criar nova coleção
  • Atualizado o menu da aplicação - adicionados termos comuns

O que há de novo no OpenNumismat 1.3.1 (05 de dezembro de 2012)

  • Reparado gravar largura das colunas da tabela.
  • Adicionado botão para abrir a o URL (secção Classificação) e inserir uma ligação a um ficheiro no disco.
  • Reparado a escolha do valor do diálogo referência ao editar a moeda.
  • Reparado formatar a descrição da coleção e campos de texto da moeda (Desenho, Assunto, etc).
  • O ficheiro da coleção pode ser passado como argumento da linha de comando.
  • Reparado possível erro ao editar comissões negativas (problema 9).
  • Adicionado apagar ficheiros temporários criados ao abrir imagens.
  • Outras alterações menores.

O que há de novo no OpenNumismat 1.3 (23 de novembro de 2012)

  • Refeito o formato da base de dados para aumentar a performance (conversão de ficheiros de coleção existentes é automática quando os abre numa nova versão do OpenNumismat).
  • Adicionados novos campos: Quantidade, URL, Código de barras.
  • Adicionada procura automática de atualizações (ativada no diálogo Definições).
  • Implementada importação de Collection Studio 3.65. Para importar, tem primeiro de exportar a coleção para uma tabela XML usando o menu Collection Studio: Ferramentas> Exportar....
  • Adicionado capacidade de proteger dados da coleção com palavra-passe (Nota: os dados não são encriptados, pelo que a palavra-passe não pode ser considerada proteção completa.) Defina a palavra-passe no menu Coleção> Definir palavra-passe....
  • Melhorada importação de CoinManage:
    • suporte para os novos campos Quantidade e Código de barras;
    • adição às moedas da coleção de imagens relacionadas com as descrições (lista de variedades);
    • importar preços.
  • Adicionado botão Reverter para predefinição no diálogo Definições para repor os nomes de todos os campos da moeda nos seus valores originais.
  • Reparado cálculo de comissões negativas para edição de informação na compra/venda das moedas.
  • Reparado deteção do idioma atual no diálogo Definições.
  • Reparado preenchimento de referências, dependendo do País dos dados da coleção.
  • Tecla Esc no diálogo Editar moeda fecha o diálogo.
  • Duplo clique na barra de separadores de lista cria uma nova lista.
  • Adicionada barra de progresso ao remover moedas.
  • Melhorada a tradução inglesa.
  • Melhorado o aspeto do diálogo Definições.
