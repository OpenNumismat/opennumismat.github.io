is een verzameling van gratis, open source munten verzamelsoftware

Catalogus voor mobiel

Voor mobiele version van OpenNumismat kan je gebruikmaken van bestaande catalogussen (beschikbare voorvertoning), geoptimaliseerd voor mobiel scherm.

Downloaden en kopiëren de catalogus naar uw apparaat:

    Om de gewenste file te selecteren, gebruik de volgende eenheden:

    • mdpi (medium) ~160dpi
    • hdpi (high) ~240dpi
    • xhdpi (extra-high) ~320dpi
    • xxhdpi (extra-extra-high) ~480dpi
    • xxxhdpi (extra-extra-extra-high) ~640dpi

    <Strong> Opmerking: Alle catalogi gebruiken informatie uit vele bronnen met verschillende licenties. Wees voorzichtig voor niet-private gebruikt. Alle materialen zijn eigendom van hun eigenaren. (Ik check it en data of vergunning te wijzigen.)
