OpenNumismat

Опис

OpenNumismat це зручний та надійний додаток, призначений для колекціонерів монет та нумізматів, для створення нумізматичної колекції.

З OpenNumismat, ви зможете організовувати і управляти колекцією монет: створювати детальний каталог ваших монет, списки потрібного і проданого, переглядати статистику вашої колекції.

Основні можливості

  • More than 80 customizable fields to describe the coin:
    • Характеристики
    • Стан монети
    • Деталі випуску
    • Данні про купівлю, продаж, прохід на аукціонах
    • Ціни та каталожні номери
    • Address and point on the map
  • Up to 11 photos of a coin (insert image from file, clipboard, download from the Web at URL)
  • Групування, фільтри та сортуваннядля полегшення пошуку монет в каталозі
  • Генерація та друк звітів, збереження у форматах HTML, PDF, MS Word
  • Експорт налаштованих списків у форматах MS Excel, HTML и CSV
  • Дублювання монет для швидкого додавання подібної монети
  • Групове редагування монет
  • Built-in image editor
  • Статистика
  • Можливість додавання та налаштування списків для відображення необхідних даних
  • Не потребує встановлення додаткових програм для роботи з базою данних
  • Імпорт монет, банкнот, штамп інформації від Colnect, онлайн-колекціонерів спільноти
  • Імпорт існуючих колекцій із
    • Excel (включаючи зображення представлені посиланнями в оригінальному файлі)
    • Colnect (from lists and directly)
    • CoinManage (required Microsoft Access Database Engine, Microsoft Access or any other software with ODBC Microsoft Access Driver)
    • Numista
    • Collection Studio 3.65 (з експортного формату XML Table)
    • uCoin.net
    • Tellico
  • Support languages: English, Russian, German, French, Ukrainian, Spanish, Hungarian, Portuguese, Greek, Czech, Italian, Catalan, Dutch, Bulgarian, Latvian, Swedish, Persian
  • Cross-platform: Windows, Linux, macOS and Android. OpenNumismat Mobile for any browser, including mobile (iPhone, Android)
  • Детальний посібник користувача
  • Suitable for other collectibles, like stamps, banknotes, postcards, badges. And also for organize home inventory

  • Export data to JSON format
  • Updated default references
  • Fixed display of the application icon in Ubuntu
  • Updated third-party libraries, compatible with newest Python 3.10

Чим я можу допомогти?

  • Використовуйте OpenNumismat та рекомендуйте його своїм знайомим.
  • Повідомляйте про помилкита нові функції, зо допоможуть покращити OpenNumismat.
  • Translate OpenNumismat and documentation to your own language with Transifex (please start with lang.ts file).
  • Якщо Ви володієте виключними правами на каталоги, фотографії або інші нумізматичні матеріали, що можуть бути включені в дані програми, зв’яжіться, будь-ласка, з автором.
  • З будь-якими запитаннями та пропозиціями можна зв'язатися з автором: opennumismat@gmail.com

OpenNumismat це:

OpenNumismat заснований на PyQt з застосуванням бази даних SQLite для зберігання даних.

Icons used in UI based on LED Icon Set v1.0, Soft Scraps Icons by deleket and Flag and Silk icons.

Всі зображення та опис монет в демо-колекціях з офіційного сайту Європейської Комісії, сайту Центрального банку Російської Федерації, Інформаційного сайту монетного двору США та Вікіпедії.
