OpenNumismat

Опис

OpenNumismat це зручний та надійний додаток, призначений для колекціонерів монет та нумізматів, створення нумізматичної колекції.

З OpenNumismat, ви зможете організовувати і управляти колекцією монет: створювати детальний каталог ваших монет, списки потрібного і проданого, переглядати статистику вашої колекції.

Знімки екранів

Більше скріншотів та відео

Основні можливості

  • Більше 80 настраюваних полів для опису монети:
    • Характеристики
    • Стан монети
    • Деталі випуску
    • Данні про купівлю, продаж, прохід на аукціонах
    • Ціни та каталожні номери
    • Адреса і відмітка на карті
  • До 11 фотографій монети (вставка зображення з файлу, буфера обміну, завантаження з Інтернету за URL)
  • Групування, фільтри та сортування для полегшення пошуку монет в каталозі
  • Генерація та друк звітів, збереження у форматах HTML, PDF, MS Word
  • Експорт налаштованих списків у форматах MS Excel, HTML и CSV
  • Дублювання монет для швидкого додавання подібної монети
  • Групове редагування монет
  • Built-in Image Editor
  • Webcam support for taking a photo or scan a barcode (for better results try Camo)
  • Статистика
  • Search by image
  • Можливість додавання та налаштування списків для відображення необхідних даних
  • Пошук та імпорт інформації про монети, банкноти, марки із Colnect, онлайн спільнота колекціонерів
  • Пошук і імпорт інформації про монети, медалі з ANS бази даних колекцій
  • Імпорт існуючих колекцій із
    • Excel
    • Colnect (зі списку)
    • Numista
    • CoinManage (потрібно Microsoft Access Database Engine, Microsoft Access або будь-яке інше програмне забезпечення з ODBC Microsoft Access Driver)
    • Collection Studio 3.65 (з експортного формату XML Table)
    • uCoin.net
    • Tellico
    • CoinSnap
  • Підтримка мов: Англійська, Російська, Німецька, Французька, Українська, Іспанська, Угорська, Португальська, Грецька, Чеська, Італійська, Каталонська, Голландська, Болгарська, Латвійська, Шведська, Перська
  • Кросплатформенність: Windows, Linux, macOS та Android. OpenNumismat Mobile для будь-якого браузера, включаючи мобільний (iPhone, Android)
  • Не потребує встановлення додаткових програм для роботи з базою данних
  • Детальний посібник користувача
  • Suitable for other collectibles, like stamps, banknotes, postcards, badges, scale models. And also for organize home inventory

Чим я можу допомогти?

  • Використовуйте OpenNumismat та рекомендуйте його своїм знайомим.
  • Повідомляйте про помилки та нові функції, що допоможуть покращити OpenNumismat.
  • Перекладайте OpenNumismat і документацію вашою рідною мовою за допомогою Transifex (почніть із файлу lang.ts).
  • Якщо Ви володієте виключними правами на каталоги, фотографії або інші нумізматичні матеріали, що можуть бути включені в дані програми, зв’яжіться, будь-ласка, з автором.
  • З будь-яких запитань та пропозицій можна зв'язатися з автором: opennumismat@gmail.com

Деталі

OpenNumismat це:

OpenNumismat based on PySide6 module with SQLite database engine to store data collection.

Піктограми, які використовуються в інтерфейсі користувача на основі LED Icon Set v1.0, Soft Scraps Icons by deleket та Flag і Silk icons.

Всі зображення та опис монет в демо-колекціях з офіційного сайту Європейської Комісії, офіційного сайту Національного банку України, інформаційного сайту монетного двору США, сайту Центрального банку Російської Федерації та Вікіпедії.
