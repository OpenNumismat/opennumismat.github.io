Каталоги для мобильного

Для мобильной версии OpenNumismat вы можете использовать существующие каталоги (доступен предпросмотр), оптимизированные для экрана мобильного.

Скачайте и скопируйте на ваше устройство любой из следующих каталогов:

    Для выбора требуемого файла используйте следующие виды плотности пикселей экрана:

    • mdpi (medium) ~160dpi
    • hdpi (high) ~240dpi
    • xhdpi (extra-high) ~320dpi
    • xxhdpi (extra-extra-high) ~480dpi
    • xxxhdpi (extra-extra-extra-high) ~640dpi

    Примечание: Все каталоги используют информацию из различных источников с различными лицензиями. Будьте осторожны при использовании не в личных целях. Все материалы принадлежат их владельцам. (Я проверю их и изменю данные или лицензию.)
