OpenNumismat

is a collection of free, open-source coin collecting software

История изменений в OpenNumismat

Новое в OpenNumismat 1.5.1 (02 февраля 2015)

  • Добавлен импорт из uCoin.net
  • Исправлен импорт из CoinManage
  • Добавлен фильтр для пустых значений в дереве
  • Улучшен диалог настройки дерева
  • Исправлена навигация по дереву при больших номиналах (> 1000)
  • Исправлены поля даты и гурта
  • Ускорено удаление монет
  • I18n

Новое в OpenNumismat 1.5 (19 Декабря 2014)

  • Обновлены PyQt до 5.3.2 и Python до 3.4.2
  • Добавлена строка поиска в диалог фильтра колонок
  • Экспорт для мобильной версии
  • Новые опции для сортировки элементов в дереве и диалоге фильтров колонок
  • Исправлено редактирование справочников, добавлена возможность добавления иконки для периода
  • Улучшено основное меню
  • Новые опции при установке: разрешение проверки обновлений и связь *.db файлов
  • Добавлен итальянский язык (от Teroknor)
  • Добавлен польский язык (от Nazzy)

Новое в OpenNumismat 1.4.9 (9 сентября 2014)

  • Добавлен перевод на чешский (от fri)
  • Исправлен debian-пакет для Ubuntu 14.04
  • Исправлена обработка поля Коммиссия

Новое в OpenNumismat 1.4.8 (6 августа 2014)

  • Добавлен перевод на немецкий (от Detlev Vogel)
  • Добавлен перевод на греческий (от geogeo)
  • Добавлен новый статус для монет - Заказана

Новое в OpenNumismat 1.4.7 (11 марта 2014)

  • Добавлен португальский перевод (от Pedro Albuquerque)
  • Исправлены некоторые ошибки

Новое в OpenNumismat 1.4.6 (10 января 2014)

  • Исправлены некоторые ошибки

Новое в OpenNumismat 1.4.5 (6 июня 2013)

  • Добавлен шаблон отчета Custom_1
  • Реализован импорт из Numizmatik_Ru
  • Обновлен импорт из Cabinet версии 2.2.2.1
  • Добавлена возможность добавлять собственные иконки флагов в справочники (правая кнопка мыши на элементе в редакторе справочников)
  • Исправлена очистка полей даты в диалоге редактирования
  • Добавлена возможность сохранения сортировки колонок (включается в диалоге Настройки)
  • Улучшена производительность сортировки и фильтрации
  • Изменена структура БД для увеличения производительности
  • Запоминается количество отображаемых изображений монеты в правой части окна
  • Улучшен автоматический скролинг дерева
  • Реализовано сохранение размеров диалогов редактирования справочников, предпросмотра отчета, настроек и т.п.
  • Исправлено поведение свободного формата для полей Цена
  • Максимальный размер изображения теперь является общим для всех коллекций

Новое в OpenNumismat 1.4.4 (2 апреля 2013)

  • Исправлены множественные ошибки при редактировании справочников
  • Добавлен венгерский перевод (от Zalán Csúszó)
  • Добавлен вывод иконки флага страны в редакторе страно-зависимых справочников
  • Добавлена команда Справка -> Проверить обновления
  • Исправлен разбор сайта Банка России
  • Небольшие улучшения шаблона Numismaster

Новое в OpenNumismat 1.4.3 (5 марта 2013)

  • Обновлена иконка приложения
  • Добавлен украинский перевод (от Алексея Ширяева)
  • Обновлен испанский перевод (от Carlos Rosi)
  • Добавлен флаг разрешения сортировке в справочниках
  • Исправлена сортировка в шаблоне cbr
  • Команда импорт из CoinCollector теперь доступна только в Windows
  • Улучшена поддержка MacOS

Новое в OpenNumismat 1.4.2 (15 февраля 2013)

  • Доступен инсталляционный пакет для Debian/Ubuntu
  • Добавлена команда Сбросить все фильтры в меню и панель инструментов
  • Добавлен шаблон отчета Numismater
  • Исправлен вывод длинных строк в некоторых полях диалога Редактирование
  • Исправлена ошибка при установке пароля
  • Кнопки сохранения в диалоге Предпросмотр отчета перемещены в левую часть панели инструментов
  • Обновлен разбор аукционов (Молоток, Wolmar)
  • Исправлено определение пути к внутренним ресурсам
  • Обновлено создание папок и файлов по умолчанию (справочник и demo.db), если отсутствуют

Новое в OpenNumismat 1.4.1 (31 января 2013)

  • Исправлена критическая ошибка с открытием приложения, когда реестр чист ( issue 12 ).

Новое в OpenNumismat 1.4 (31 января 2013)

  • Реализована генерация отчетов (на основе шаблонов Jinja2).
  • Реализован экспорт настроенного списка (Отчет > Сохранить текущий список...).
  • Добавлена возможность вводить любые данные в числовые поля (необходимо разрешить Свободный формат числовых полей в диалоге Настройки) ( issue 6 ,  issue 10 ).
  • Исправлена автоматическая проверка обновлений (при частом запуске приложения проверка не осуществлялась).
  • Добавлена команда Сжать (меню Коллекция) освобождающая место на диске, зарезервированное движком БД.
  • Изменено предлагаемое имя файла при сохранении изображений монеты в файл.
  • Исправлена возможная ошибка при открытии файла, указанного в поле URL описания монеты.
  • Исправлена возможная ошибка при создании новой коллекции.
  • Обновлено меню приложения - добавлены некоторые часто используемые пункты.
  • Не выводятся пункты меню импорта, при недоступности модулей Python (при запуске из исходных кодов).

Новое в OpenNumismat 1.3.1 (05 декабря 2012)

  • Исправлено сохранение ширины выводимых столбцов таблицы.
  • Добавлены кнопки открытия URL (раздел Классификация) и вставки ссылки на файл на диске.
  • Исправлен выбор значения из диалога справочника при редактировании монеты.
  • Справочник Качество чеканки переведен на русский язык.
  • Исправлено форматирование описания коллекции и текстовых полей описания монеты (Дизайн, Тема и т.д.).
  • Файл коллекции может быть передан как аргумент командной строки.
  • Исправлена возможная ошибка при редактировании отрицательных комиссий ( issue 9 ).
  • Добавлено удаление временных файлов, создаваемых при открытии изображений.
  • Другие мелкие изменения.

Новое в OpenNumismat 1.3 (23 ноября 2012)

  • Переработан формат базы данных для увеличения производительности (конвертирование существующих файлов коллекций происходит автоматически при их открытии в новой версии OpenNumismat).
  • Добавлены новые поля: Количество, URL, Штрих-код.
  • Добавлена автоматическая проверка наличия обновлений (включается в диалоге Настройки).
  • Реализован импорт из Collection Studio 3.65. Для импорта необходимо предварительно экспортировать существующую коллекцию в формат XML Table с помощью меню Collection Studio: Инструменты > Экспорт....
  • Добавлена возможность защитить данные коллекции паролем (Внимание: шифрование данных не осуществляется, поэтому защиту паролем нельзя считать полноценной). Установка пароля осуществляется в меню Коллекция > Установить пароль....
  • Улучшен импорт из CoinManage:
    • поддержка новых полей Количество и Штрих-код;
    • добавление к монетам из коллекции изображений связанных описаний (variety list);
    • импорт цен.
  • Добавлена кнопка Восстановить по умолчанию в диалоге Настройки для восстановления названий всех полей описания монеты к исходному состоянию.
  • Исправлено вычисление отрицательных комиссий при редактировании информации о покупке/продаже монеты.
  • Исправлено определение текущего языка в диалоге Настройки.
  • Исправлено заполнение справочников, зависящих от страны, данными из коллекции.
  • Нажатие клавиши Esc в окне редактирования монеты приводит к закрытию окна.
  • Двойной щелчек в панели вкладок списков создает новый список.
  • Добавлен вывод индикатора выполнения при удалении монет.
  • Улучшен английский перевод.
  • Улучшен вид окна Настройки.
