OpenNumismat

История изменений в OpenNumismat

Что нового в OpenNumismat 1.5.4 (28 сентября 2017)

  • Added exchanging images in Edit coin dialogue (by right-click)
  • Don't change collection file when nothing changed (useful for Dropbox, etc)
  • Added import from Tellico
  • Added Catalan, Dutch translations. Update Greek, Ukrainian translations
  • Added settings for protect extra warnings on save
  • Fix error report
  • Fix pasting in numeric fields
  • Show collection name when entering password
  • Max image side len 0 for storing in original size
  • Enabled inserting *.ico as reference item icon
  • Improved getting boolean settings on MacOS
  • Other minor improvements

Что нового в OpenNumismat 1.5.3 (19 февраля 2016)

  • Исправлена потеря курсора при вставке новых записей

  • Исправлена фильтрация колонки Статус
  • Добавлена кнопка Переименовать в настройки
  • Исправлен ввод чисел для некоторых романских языков
  • Улучшен ввод значений в поле Тираж
  • Исправлена вставка изображений из браузера
  • Исправлена очистка полей Дата
  • Добавлен перевод на французский (от Mathieu Martineau, Jean-Claude Kursner)
  • Добавлена опция для сжатия изображений при импорте в мобильную версию
  • Обновлена иконка очистки всех фильтров
  • Updated additional libraries and dependencies
