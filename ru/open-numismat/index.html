OpenNumismat

Описание

OpenNumismat является удобным и надежным приложением, предназначенным для коллекционеров монет и нумизматов, для создания нумизматической коллекции.

С OpenNumismat вы сможете организовывать и управлять коллекцией монет: создавать подробный каталог ваших монет, списки нужного и проданного, просматривать статистику вашей коллекции.

Основные возможности

  • Более 80 настраиваемых полей для описания монеты:
    • Характеристики
    • Состояние монеты
    • Детали выпуска
    • Данные о покупке, продаже, проходе на аукционах
    • Цены и каталожные номера
    • Адрес и точка на карте
  • До 11 фотографий монеты (вставка изображения из файла, буфера обмена, загрузка из Web по URL)
  • Группировка, фильтры и сортировка для облегчения поиска монет в каталоге
  • Генерация и печать отчетов, сохранение в форматах HTML, PDF, MS Word
  • Экспорт настроенных списков в форматы MS Excel, HTML и CSV
  • Дублирование монет для быстрого добавления сходной монеты
  • Групповое редактирование монет
  • Встроенный редактор изображений
  • Статистика
  • Возможность добавления и настройки списков для отображения требуемых данных
  • Не требует установки дополнительных программ для работы с базой данных
  • Поиск и импорт информации о монетах, банкнотах, марках из Colnect, онлайн-сообщества коллекционеров
  • Поиск и импорт информации о монетах из базы коллекцийANS
  • Импортирование существующих коллекций из
    • Excel (включая изображения представленные ссылками в оригинальном файле)
    • Colnect (из списков)
    • Numista
    • CoinManage (требуется Microsoft Access Database Engine, Microsoft Access или любое другое приложение с ODBC Microsoft Access Driver)
    • Collection Studio 3.65 (из экспортного формата XML Table)
    • uCoin.net
    • Tellico
  • Поддержка языков: русский, украинский, английский, испанский, немецкий, французский, венгерский, португальский, греческий, чешский, итальянский, каталонский, нидерландский, болгарский, латвийский, шведский, персидский
  • Кросс-платформенность: Windows, Linux, macOS и Android. OpenNumismat Mobile для любых браузеров, включая мобильные (iPhone, Android)
  • Подробное руководство пользователя
  • Подходит для других типов коллекционирования, например марки, банкноты, открытки, значки

Как я могу помочь?

  • Используйте OpenNumismat и рекомендуйте его своим знакомым.
  • Сообщите о найденных ошибках и новых функциях, которые могут сделать OpenNumismat лучше.
  • Переведите OpenNumismat и документацию на ваш язык с помощью Transifex (начните с файла lang.ts).
  • Если вы обладаете исключительными правами на каталоги, фотографии или иные нумизматические материалы, которые могут быть включены в данные программы, пожалуйста, свяжитесь с автором.
  • С любыми вопросами и предложениями вы можете обратиться к автору: opennumismat@gmail.com

Детали

OpenNumismat это:

OpenNumismat основывается на библиотеке PyQt с SQLite database engine для хранения данных коллекции.

Иконки, используемые в интерфейсе, основываются на LED Icon Set v1.0, Soft Scraps Icons от deleket и Flag и Silk icons.

Все изображения и описания монет в демо-коллекциях с официального сайта Европейской Комиссии, сайта Центрального банка Российской Федерации, информационного сайта монетного двора США и Википедии.
