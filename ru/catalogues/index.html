OpenNumismat

is a collection of free, open-source coin collecting software

Каталоги

Для начала работы с программой OpenNumismat можно использовать уже созданные коллекции, содержащие информацию о выпущенных монетах (каталоги).

Для скачивания доступны следующие каталоги монет:

    • standard coins of EU (English) 24.04.2013
    • commemorative Ukrainian (Ukrainian) 19.02.2013
    • all Russian coins (Russian) (commemorative and bullion coins 1992-2013) 08.04.2013

    Примечание: Все каталоги используют информацию из различных источников с различными лицензиями. Будьте осторожны при использовании не в личных целях. Все материалы принадлежат их владельцам. (Я проверю их и изменю данные или лицензию.)

    Примечание: В каталогах на английском языке все поля также имеют английские наименования. Для перевода наименований полей на русский язык необходимо в диалоге Настройки во вкладке Поля нажать кнопку Восстановить по умолчанию.
