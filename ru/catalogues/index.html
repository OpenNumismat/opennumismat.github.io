Каталоги

Импортом из Colnect (из списков и напрямую) вы можете быстро создать собственный каталог.

Для начала работы с программой OpenNumismat можно использовать уже созданные коллекции, содержащие информацию о выпущенных монетах (каталоги).

Для скачивания доступны следующие каталоги монет:

    • стандартные монеты ЕС (английский) 24.04.2013
    • Евро монеты 1999 - 2014 (английский)
    • Стандарные монеты США (английский) 20.03.2024
    • Польские монеты (польский)
    • памятные монеты Украины (украинский) 12.02.2023
    • памятные монеты России 1992-2013 (русский) 08.04.2013
    • демнстрационная коллекция марок (английский)
    • демонстрационная коллекция марок (русский)
    • демонстрационная коллекция открыток (английский)
    • памятные монеты 2 евро (испанский)
    • Монеты Испании (испанский)
    • монеты Хуана Карлоса I (испанский)
    • монеты Эстонии (испанский)
    • монеты Болгарии 1881-1943 (болгарский)

    Примечание: Все каталоги используют информацию из различных источников с различными лицензиями. Будьте осторожны при использовании не в личных целях. Все материалы принадлежат их владельцам. (Я проверю их и изменю данные или лицензию.)

    Примечание: В каталогах на другом языке все поля также имеют отличные наименования. Для перевода наименований полей на русский язык необходимо в диалоге Настройки во вкладке Поля нажать кнопку Восстановить по умолчанию.
