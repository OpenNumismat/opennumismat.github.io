OpenNumismat для Android

Описание

OpenNumismat приложение для коллекционеров монет.

С этим приложением ваша коллекция монет всегда в ваше устройстве Android. Вы можете использовать существующие каталоги или создать свой собственный с помощью настольной версии OpenNumismat.

Это мобильная версия OpenNumismat, которая поможет отслеживать монеты, которые у вас уже есть.
