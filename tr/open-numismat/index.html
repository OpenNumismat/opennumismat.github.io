OpenNumismat

Açıklama

OpenNumismat, madeni para koleksiyoncuları, nümizmatlar ve para koleksiyonu yapmak isteyen amatörleri hedefleyen kullanışlı ve güvenilir bir uygulamadır.

OpenNumismat ile madeni para koleksiyonunuzu düzenleyebilecek ve yönetebileceksiniz: gelişmiş bir katalog oluşturabilir, istek ve satış listeleri yapabilir, koleksiyonunuzun istatistiklerini görüntüleyebilirsiniz.

Ana Özellikler

  • More than 80 customizable fields to describe the coin:
    • Özellikler
    • Madeni paranın durumu
    • Emisyon Detayları
    • Buy, sell, pass on auction information
    • Fiyatlar ve katalog numaraları
    • Address and point on the map
  • Up to 11 photos of a coin (insert image from file, clipboard, download from the Web at URL)
  • The grouping, filters and sorting to facilitate the retrieval of coins in the catalog
  • Generate and print reports, saving as HTML, PDF, MS Word
  • Export customized lists as MS Excel, HTML and CSV
  • Duplication of coins to quickly add a similar coin
  • Batch edit coins
  • Built-in image editor
  • İstatistikler
  • Ability to add and customize the lists to display the required data
  • Does not require additional software to work with a database
  • Çevrimiçi bir koleksiyoncu topluluğu olan Colnect'ten madeni para, kağıt para ve pul bilgilerini içe aktarma
  • Var olan koleksiyonları içe aktarma:
    • Excel (with images presented as links in source file)
    • Colnect (from lists and directly)
    • CoinManage (required Microsoft Access Database Engine, Microsoft Access or any other software with ODBC Microsoft Access Driver)
    • Numista
    • Collection Studio 3.65 (from a export format XML Table)
    • uCoin.net
    • Tellico
  • Desteklenen diller: İngilizce, Rusça, Almanca, Fransızca, Ukraynaca, İspanyolca, Macarca, Portekizce, Yunanca, Çekçe, İtalyanca, Katalanca, Flemenkçe, Bulgarca, Letonca, İsveççe, Farsça
  • Cross-platform: Windows, Linux, macOS and Android. OpenNumismat Mobile for any browser, including mobile (iPhone, Android)
  • Detaylı kullanım kılavuzu
  • Suitable for other collectibles, like stamps, banknotes, postcards, badges. And also for organize home inventory

Detaylar

OpenNumismat :

OpenNumismat based on PyQt framework with SQLite database engine to store data collection.

Icons used in UI based on LED Icon Set v1.0, Soft Scraps Icons by deleket and Flag and Silk icons.

All images and descriptions of coins in demo collections from the Official website of the European Commission, The Central Bank of the Russian Federation website, United States Mint's public information Website and Wikipedia.
